🏕️ Hackathon

SETU – AI Lost & Found System

Developed during the Claude Impact Lab Hackathon, SETU is an AI-powered lost-and-found platform designed for the Trimbakeshwar Kumbh Mela. The system uses intelligent matching, geolocation and multilingual support to reconnect missing people and belongings while improving crowd management during large-scale public events.

  • React

StudyIQ – Smart Assignment Tracker

StudyIQ is an intelligent assignment management platform built using Python and Streamlit. It features secure email OTP authentication, subject-wise assignment organization, file uploads, automated priority calculation, deadline tracking and an integrated Google Gemini AI assistant.

  • Python
  • Streamlit
  • SQLite
  • Gemini API
