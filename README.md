# 📖 Project Overview

This project is a professional dark-themed personal portfolio consisting of multiple interconnected pages. It demonstrates modern front-end development practices including semantic HTML, responsive web design, accessibility, animations and JavaScript-powered interactivity.

The website is completely built without any frameworks or libraries, making it lightweight, fast and easy to understand.

# 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

Navigate into the project.

```bash
cd portfolio
```

Open **index.html**

or

Run a local server.

```bash
python3 -m http.server 8000
```

or

```bash
npx serve .
```

Visit

```
http://localhost:8000
```
